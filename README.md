# JSONParser

### JSON 解释器

使用原生 JavaScript 实现，能把 JSON 文本解析成 JavaScript 数据结构

### 项目结构

```
|   .gitignore
|   main.js         // JSONParser
|   README.md
|
\---lib
        test.js     // 测试用例
        util.js     // 工具函数
```

### 启动

```sh
git clone
```
```sh
cd ./JSONParser/lib
```
```sh
node test.js
```
